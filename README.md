## 大会開催情報

* 大会ウェブページ / Official page: https://www.robocup.or.jp/JapanOpen2025/
* ＠ホームリーグウェブページ / @Home league page: https://sites.google.com/site/robocuphomejapan/japanopen2025

* 会場 / Location: 滋賀ダイハツアリーナ / Shiga Daihatsu Arena, Japan
